# NotesApp-backend

### The backend for the NotesApp using :

- `Typescript`
- `Express` for handling API request
- `Sqlite` as database
- JWT or Session-based Authentication

### `.env` File Format

```bash
ACCESS_TOKEN='<string>'
MONGO_USER='<string>'
MONGO_PASSWORD='<string>'
MONGO_CLUSTER='cluster0.<string>.mongodb.net/?retryWrites=true&w=majority'
```
